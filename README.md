# Outfit Oracle

Outfit Oracle is an AI-powered wardrobe assistant that generates outfit recommendations from a real wardrobe inventory.

Users enter an occasion, temperature, weather, and vibe. The app recommends a complete outfit using only items from the available wardrobe data.

## Features

- AI-generated outfit recommendations
- Uses a structured wardrobe dataset
- Considers occasion, weather, temperature, vibe, color, fabric, and formality
- React frontend
- Express/Node backend
- OpenAI API integration

## Tech Stack

- React
- Vite
- Node.js
- Express
- OpenAI API
